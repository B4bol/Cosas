Recurso Descripcion Ejemplo
class Con el argumento class podremos cambiar lo que queramos de forma especifica se aplica dentro de las etiquetas de HTML class="ejemplo"
background-color cambia el color de fondo en el class especificado background-color: "[codigo de color en hexadecimal, nombre o rgb]";
text-aling Sirve para alienar el texto en una direcceion como a la izquierda o al centro text-aling: "center";
color Podemos cambiar el color que no es el fondo de la clase seleccionada color: "[codigo de color en hexadecimal, nombre o rgb]";
width Podemos cambiar la anchura de algo con este argumento width: [x]px;
height Podemos cambiar la altura de algo con este argumento height [x]px;
