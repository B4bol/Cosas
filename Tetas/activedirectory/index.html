ACTIVE DIRECTORY

Active Directory es un servicio de directorio desarrollado por Microsoft para redes de dominios
Se incluye en la mayoría de los sistemas operativos Windows Server como un conjunto de procesos y servicios.
Inicialmente, Active Directory solo se encargaba de la administración centralizada de dominios.
Sin embargo, Active Directory se convirtió en un título general para una amplia gama de servicios
relacionados con la identidad basados en directorios.
